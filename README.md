# stayhome-aging-in-place

A local-first product scaffold for a senior home-safety audit and contractor matching platform focused on helping families reduce fall risk and support aging in place.

## Stack
- Next.js (App Router) + TypeScript
- Bun
- Tailwind CSS

## Getting Started
1. Install dependencies:
   `bun install`
2. Copy env template:
   `cp .env.example .env.local`
3. Run local dev server:
   `bun run dev`

## Scripts
- `bun run dev` - start development server
- `bun run build` - production build
- `bun run start` - run production server
- `bun run lint` - eslint
- `bun run typecheck` - TypeScript type check
- `bun run test` - unit test runner

## Notes
- This scaffold includes planning docs (`IMPLEMENTATION_PLAN.md`, `progress.txt`) for implementation handoff.
- No production business logic is implemented yet.
